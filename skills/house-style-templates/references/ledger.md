# Ledger — The Engraved Prospectus

**Identity (one line):** An engraved heritage certificate / private-bank prospectus — security-print craft plus fine editorial; the document you are *handed*, not shown.

**Register:** EXPRESSIVE-BUT-RESTRAINED — quiet authority; **the confidence IS the restraint.** It reads the way old money speaks: certain, unhurried, nothing to prove. Mood: opulent, calm, exact, permanent.

**Not accounting.** "Ledger" is a slot label only — no folio `№`, no double-entry, no running-balance metaphors. That reading was tried and killed; keep it dead.

> **This is a baseline, not the design.** A template sets the starting register, palette, and move-set; the actual design is expected to diverge from it wherever the user's brief, content, or audience demands. See "Baselines, not cages" in SKILL.md.

## Pull it when

- Prospectuses, offering documents, annual letters
- Wealth, heritage, and private-bank identities
- Certificates, credentials, instruments of record
- Editorial reports meant to be *kept*, not skimmed
- High-trust financial and legal surfaces
- Any page whose value is that it looks **permanent**

## Wrong choice for

- Playful consumer apps, toys, games (→ `prism.md`)
- High-density dashboards that must flash loud, fast signals (→ `terminal.md`)
- Fast-moving SaaS that re-skins itself each quarter
- Discount, flash-sale, growth-hacky surfaces — anything that must feel cheap, urgent, or casual
- Interfaces where scanning speed must beat gravitas (→ `meridian.md`)

## The vocabulary — colour

| Token | Value | Role |
|---|---|---|
| `--paper` | `#efe9da` | cream laid stock |
| `--ink` / `--ink-soft` | `#20201b` / `#3a382f` | near-black engraving ink |
| `--accent` | `#17533d` | THE one deep-green functional accent — spent only where a decision is made |
| `--brass` | `#8a7338` | **foil material only** — rules, marks, large elements, ≥3:1; never voice, never small text |
| `--rule` / `--panel` | `#d6cdb8` / `#e8e1cf` | hairlines, panels |
| `--pop` | `#c0362c` | cinnabar — the ONE contrasting pop, spent exactly once page-wide |

## The vocabulary — type

- **Display:** Instrument Serif — high-contrast marquee, pressed into the stock
- **Text:** Plus Jakarta Sans
- **Figures:** IBM Plex Mono, tabular throughout

## The vocabulary — structure (the fixed move-set)

- **Hairline rules holding every column true**; a felt ghost-grid
- **Guilloché rosette seal** where a maker's mark belongs (dense at hero scale; sheds the guilloché at small sizes so the mark stays fluent)
- **Engraved plates** — drawn, two-ink duotone (two inks far apart in value); never glossy photography. If the brief requires photographs (portraits, premises), they enter as treated, toned plates in the engraved hand — a noted divergence, not a register break
- **Roman-numeral enumerator** binding sections into one numbered specimen set
- **Rare brass act-seams** — two or three page-wide, marking true act changes; scarcity is what makes them read

## Held constants (the core — what must survive for it to still be Ledger)

Cream laid stock · the engraved two-ink hand · hairline rules holding columns true · restraint-as-argument register (every mark earns its ink). Everything else — the seal's density, brass seams, the enumerator, even the accent hue — is swappable within the floors.

## Discipline

- The fewer marks a page spends, the more each is worth — restraint is the argument, so every added element must *earn* its ink.
- One cinnabar pop live once; labeled palette-teaching specimens are documentation, not a second emphasis.
- All four floors hold (AA measured, chart honesty, legibility, focus/44px). The look bends to the rule, always.
