#### Source `aa0ba4e5` — quality: **primary** · published: 2013
- **[central]** A distinctive style/innovation is developed via PAIRED CONSTRAINTS: each pair has one constraint that precludes (rules out) an element and a second that specifies its substitute; the process can be self-imposed ('by design') or externally forced ('by necessity'). This is the paper's core, teachable mechanism for deliberately constructing a style.
  > "One of each pair identifies something to be precluded; the other specifies its substitute. The process can be initiated by necessity or by design."
- **[central]** The recognizable creation/style is not chosen wholesale but EMERGES as the novel solution path produced by an iterative, step-by-step SERIES of substitutions (each substitution necessitating the next). Style is the cumulative product of the substitution chain, not a single act.
  > "The creation/innovation is the novel solution path created by the substitution series."
- **[central]** The first step in originating a new style is deliberate DEVIATION FROM CONVENTION: precluding specific elements of an existing, well-defined ('well-structured') conventional style, which converts it into an 'ill-structured' problem that now requires novel substitutions.
