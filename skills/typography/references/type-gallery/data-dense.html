Dense data — the modernist kit under pressure

The hardest test of data typography: compact many-column tables, program/project management, and quant data science, all in the default kit (Bricolage Grotesque headings · Plus Jakarta Sans labels · IBM Plex Mono numbers) on Corbusier grounds. The discipline that makes dense data legible is mono tabular figures (every digit one width → columns align and scan), small but ≥ readable sizes, quiet labels, right-aligned numbers, and one reserved accent / RAG status by colour and glyph — the typography, colour, charting and UX skills composing under load.
